Business Logic Policy

El frontend nunca realiza cálculos de negocio.

El frontend solo captura datos,

los envía a Supabase y presenta los resultados.

Toda validación,

cálculo,

regla de negocio

y transformación de datos

debe ejecutarse en Supabase.
