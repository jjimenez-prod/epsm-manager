# Versioning Policy

EPSM Manager follows Semantic Versioning.

Format:

MAJOR.MINOR.PATCH

Example:

1.0.0

Where:

- MAJOR → Breaking architectural or functional changes.
- MINOR → New features.
- PATCH → Bug fixes and small improvements.
